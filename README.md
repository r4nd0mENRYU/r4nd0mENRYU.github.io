# r4nd0mENRYU GitHub Pages Portfolio

정적 GitHub Pages 포트폴리오입니다. 빌드 과정 없이 `index.html`을 바로 배포할 수 있습니다.

## 배포 방법

1. GitHub에서 `r4nd0mENRYU.github.io` 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소 루트에 커밋합니다.
3. 저장소 Settings → Pages에서 `main` 브랜치와 `/root`를 선택합니다.
4. 배포 후 주소는 `https://r4nd0menryu.github.io` 형태로 열립니다.

## 수정할 곳

- 프로젝트 카드: `index.html`
- 화면 스타일: `styles.css`
- 필터 동작: `script.js`
- 첫 화면 이미지: `assets/portfolio-hero.png`
